# NPM Registry

A React application for searching and exploring npm packages. This project provides a user-friendly interface to interact with the npm registry, allowing users to discover, search, and view detailed information about packages directly from the npm ecosystem.

## Project Description

This application enables users to:

- Search for npm packages in real-time using the npm registry API.
- Browse search results with essential package information such as name, latest version, and short description.
- Click on any package to view comprehensive details, including full description, version history, maintainers, repository links, and download statistics.
- Experience a responsive and intuitive UI designed for both desktop and mobile devices.
- Learn how to integrate public APIs, manage asynchronous data fetching, and build modern React interfaces.

## Features

- Search for npm packages in real-time
- View package details including version, description, and metadata
- Display package statistics and download information
- Responsive design for mobile and desktop viewing
- Integration with the npm registry API

## Technologies Used

- React
- JavaScript/TypeScript
- API Integration (npm registry)
- Tailwind CSS for styling

## Getting Started

### Prerequisites

- Node.js and npm installed

### Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:5173) to view it in the browser

## Usage

Enter a package name in the search bar to find packages in the npm registry. Click on a package to view more detailed information.
